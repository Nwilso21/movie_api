For all of the following endpoints, the data input and output will be in JSON format

Request URL Response
GET localhost:8080/movies returns a list of all movies
GET localhost:8080/movies/movieTitle returns all store data on the given movie
GET localhost:8080/movies/genre/wantedGenre returns info about specified genre
GET localhost:8080/movies/director/movieTitle returns the director of the movie
POST localhost:8080/userList Adds a new user to the list
PUT localhost:8080/user/userID changes user details
POST localhost:8080/users/userID/movieTitle adds the specified movie to the specified users favorites list
DELETE localhost:8080/users/userID/movieTitle removes the movie from the specified users favorites list
DELETE localhost:8080/users/userID deletes the given user from the user list
